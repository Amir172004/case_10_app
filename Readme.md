# 📉 Crash-O-Meter — Аналитический дашборд по краху рынка

Веб-приложение на Python (Flask) для визуализации противоположных мнений аналитиков о вероятности краха американского фондового рынка.

**Особенности:**
- Интерактивные слайдеры (CAPE, кэш Баффета)
- Кнопки быстрого выбора сценариев (бычий/медвежий/случайный)
- Динамическая цитата аналитика
- График «индекса рыночного безумия» (Chart.js)
- Вероятность краха в реальном времени

---

## 📋 Требования

- **macOS** (или любая ОС с Python)
- **Python 3.9.4** или выше
- **pip** (менеджер пакетов Python)
- **Браузер** (Chrome, Safari, Firefox и т.д.)
